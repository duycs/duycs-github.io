Command line for Mac, Linus, Windows 

## I. Vim
#### 1. Moving 
- left
```H```
- down 
```J```
- up 
```K```
- right
```L```
- end of line
```$```
- top of line
```0```
- top of file
```gg```
- end of file
```G```
- line at 50
```50G```

#### 2. Mode
- insert mode
```i```
- replace mode
```R```
- delete a position at pointer
```x```

#### 3. Save, exit
- exit
```:q```
- exit wihout save
```:q!```
- save
```:w```
- save override
```:w!```
- save then exit
```:wq```
